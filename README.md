# SolutionTap



This is an open source project made under the emergence of Covid-19 pandemic as the solution and one of the solution in Hackon Hackathon.

Technologies/Techstack Used
* Firebase
* Java 
* XML
* JSON

Modules
* List of patients with details
* Resolved Queries
* Display Videos
